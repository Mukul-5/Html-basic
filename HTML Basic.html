6. Semantic HTML5 Elements

These tags give meaning to the structure (important for SEO & accessibility):

  • <header> - Introductory content or navigational links
  • <nav> - Navigation menu
  • <main> - The main content of the document
  • <article> - Independent, self-contained content (like a blog post)
  • <section> - A thematic grouping of content
  • <aside> - Content aside from the content (like a sidebar)
  • <footer> - Footer of a document or section

8. Containers (Div & Span)

<div> is a block-level container. It takes up the full width available.

<span> is an inline container. It only takes up as much width as necessary and does not start on a new line.

9. Iframes

Iframes are used to embed another webpage within the current HTML document.
